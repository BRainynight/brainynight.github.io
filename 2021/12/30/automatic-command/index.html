說明

先開啟網頁是因為一旦 server 打開就會佔一個 process 在那邊，阻斷後面的指令。而網頁先開啟來只是一個動作，不會阻斷後續的指令。

local

是每次改完部落格，要清除靜態檔案重新生成並打 local 的展示網頁的時候。

demo

單純想打開 local demo，沒有要清掉資料重新生成。

push
